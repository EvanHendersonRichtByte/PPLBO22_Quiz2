# Sistem Penilaian Mahasiswa
### Alat untuk membantu para dosen dalam melakukan input data mahasiswa
#### Sistem ini dikembangkan menggunakan bahasa pemrograman [Java](https://www.java.com/en/)
### Fitur
- CRUD Mahasiswa
- CRUD Nilai
- CRUD Kelas
- CRUD Mata Kuliah
### Fitur Kedepan
- CRUD Dosen
- CRUD DPA
